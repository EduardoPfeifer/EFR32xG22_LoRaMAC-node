# Port of Semtech LoRaMAN-node to Silicon Labs EFR32xG22 wireless SoC #

## IDE
Project developped using Simplicity Studio 5 with BLE SDK v3.0.0.2

## Connections ##

| SX1261 pin | WSTK EXP pin | EFR32MG22 (QFN40) pin |
|------------|--------------|-----------------------|
| SCK        | 6            | PC02                  |
| MOSI       | 4            | PC00                  |
| MISO       | 6            | PC01                  |
| NSS        | 10           | PC03                  |
| DIO1       | 12           | PA05                  |
| BUSY       | 14           | PA06                  |
| RESET      | 16           | PB03                  |
